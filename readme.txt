Easily enable a customizable maintenance mode page for your WordPress site with personalized messages and designs.

== Description ==

**Site Standby** allows you to create a customizable maintenance mode page to keep your visitors informed while you update your website. This plugin is perfect for scheduling maintenance or making quick updates without losing your audience.

**Features:**
* Custom Message: Display a personalized message to your visitors.
* Background Color: Choose a background color that matches your branding.
* Custom HTML Content: Add any HTML content to your maintenance page for a unique touch.
* Custom CSS: Optionally link custom CSS styles to further personalize your maintenance page.
* Admin Access: Only administrators can bypass the maintenance mode and access the site.

== Installation ==

1. Upload the `site-standby` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to Settings -> Site Standby to customize your maintenance mode settings.

== Frequently Asked Questions ==

= How do I customize the maintenance mode page? =

Navigate to Settings -> Site Standby in your WordPress admin dashboard. Here, you can set a custom message, background color, and even add your own HTML content.

= Who can bypass the maintenance mode? =

Only users with administrator privileges can bypass the maintenance mode and access the site normally.
